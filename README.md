# Vibe - Music Player

Vibe is a simple web-based music player project built using HTML, CSS, and JavaScript. It starts from `landing.html` and moves to `new.html`, which is the main player page.

## Introduction
A music player is a software application designed to play audio files and let users listen to songs easily. It usually supports basic controls like play, pause, next, previous, and playlist browsing. Vibe is a small student project made to keep the design simple and easy to understand.

## Key Features
- Simple landing page with a short intro about Vibe.
- Music player page with song categories and playback controls.
- Easy navigation from `landing.html` to `new.html`.
- Basic and clean UI made for a first-semester style project.

## How it Works
1. Open `landing.html`.
2. Click the `Open Music Player` button or use the navbar link.
3. The page opens `new.html`, which contains the music player.

## Project Pages
- `landing.html` - landing page with project overview and key features.
- `new.html` - music player page with categories, songs, and controls.
