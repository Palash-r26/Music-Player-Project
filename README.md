# Vibe - Music Player

Vibe is a simple music player project built with HTML, CSS, and JavaScript. The project uses `landing.html` as the start page and opens `new.html` as the player page.

## How it works
1. Open `landing.html`.
2. Use the `Open Music Player` button or the navbar music link.
3. You will go to `new.html`, which contains the music player UI.

## Pages
- `landing.html` - simple landing page with project intro and key features.
- `new.html` - music player page with categories, songs, and playback controls.

## Notes
- The repository does not use `index.html` anymore.
- The UI is kept simple and consistent for a first-semester style project.
