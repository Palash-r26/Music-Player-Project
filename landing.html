What is Vibe?

Vibe is a simple, student-friendly music player web project that lets you browse categories and play songs in a lightweight UI. It focuses on clarity and ease-of-use — suitable for learning web development.

Simple UI: Clear layout with categories and a compact player.
Playlists: Browse songs by category and play instantly.
Controls: Play/Pause, Next/Prev and Repeat support.
Lightweight: Built with basic HTML, CSS and JavaScript.
